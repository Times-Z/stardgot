## Project Structure

- `Scenes/` : Main game scenes (maps, menus, player, etc.)
- `Scripts/` : C# scripts for game logic
- `assets/` : Art, music, and sound resources
- `TileSet/` : Tilesets for map creation
- `project.godot` : Godot project configuration

---

## How to Run

1. Open the project folder in **Godot 4.4 (Mono)**.
2. Make sure you have .NET SDK 8.0 installed.
3. Click **Play** in the Godot editor to start the game.
